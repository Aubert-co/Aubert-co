# Aubert Barbosa

Desenvolvo aplicações com foco em backend, testes automatizados, arquitetura de APIs, containers e integração com serviços AWS.

Refatorei um e-commerce pessoal apoiado por mais de 500 testes automatizados, cobrindo fluxos de autenticação, produtos, lojas, carrinho, cupons, upload de imagens e cenários de erro.

Construí projetos com Node.js, TypeScript, React, PostgreSQL, MySQL, Redis, Docker, Docker Compose, AWS S3, AWS Lambda, Grafana, Loki e Jest.

## Como eu trabalho

* Planejei funcionalidades antes de expandir módulos.
* Implementei testes automatizados para sustentar refatorações.
* Modelei APIs com separação entre controller, service, repository e providers.
* Isolei integrações externas para reduzir acoplamento.
* Configurei ambientes com Docker e Docker Compose.
* Integrei serviços AWS em fluxos de upload e processamento de imagens.
* Registrei logs estruturados para análise em ambiente de observabilidade.
* Documentei decisões técnicas nos projetos do portfólio.

## Destaques

* Implementei mais de 500 testes automatizados com Jest.
* Desenvolvi APIs REST com Node.js, NestJS e TypeScript.
* Integrei upload de imagens com AWS S3.
* Desacoplei processamento de imagens usando AWS Lambda.
* Configurei ambientes locais com Docker e Docker Compose.
* Configurei observabilidade com Grafana, Loki e Grafana Alloy.
* Estruturei cache com Redis.
* Modelei bancos relacionais com PostgreSQL,  Prisma.

## Portfólio

Documentei meus principais projetos no portfólio online, incluindo arquitetura, testes, observabilidade, containers, integrações AWS e decisões de implementação.
